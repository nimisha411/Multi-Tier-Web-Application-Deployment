# 🚀 Multi-Tier Web Application Deployment on AWS

## 📌 Project Overview
This project demonstrates deployment of a Multi-Tier Web Application on AWS Cloud.

The application architecture is divided into three layers:

- Frontend Layer
- Backend Layer
- Database Layer

This architecture improves scalability, availability, and security of the application.

---

# 🧰 AWS Services Used

- Amazon EC2
- Amazon RDS
- Application Load Balancer (ALB)
- Security Groups

---

# 💻 Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- MySQL
- AWS Cloud

---

# 🏗️ Architecture

Users → Application Load Balancer → EC2 Frontend → EC2 Backend → Amazon RDS
